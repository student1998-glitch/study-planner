# Task Planner — Product Concept

## Problem
Багато студентів і зайнятих користувачів мають труднощі з організацією
повсякденних справ і дедлайнів. Завдання часто губляться або забуваються,
що призводить до стресу та зниження продуктивності.

## Pitch
Task Planner — це простий цифровий планувальник для студентів та
активних користувачів, які хочуть швидко фіксувати задачі та контролювати
їх виконання. Продукт допомагає впорядкувати справи в одному місці та
зосередитись на головному результаті — вчасно виконаних задачах.

## MVP (Core Idea)
Мінімально життєздатний продукт зосереджується на одному ключовому
сценарії: користувач може створити задачу, побачити її в основному
списку та позначити як виконану. Без реалізації цього сценарію цінність
продукту не досягається.

## Stakeholders (Narrative Map)
Кінцеві користувачі (студенти) отримують основну цінність у вигляді
зручного інструменту для планування. Викладач або наставник оцінює
досягнення цілей через покращення організованості студентів. Розробник
впливає на реалізацію продукту з огляду на обмеження часу та технічних
ресурсів.

## First Iteration Goal
Метою першої ітерації є реалізація головного сценарію управління задачами
без зайвих кроків і складних налаштувань.

## Success Criteria
- Користувач може створити та завершити задачу за мінімальну кількість
  кроків.
- Основні операції (створення та завершення задач) працюють стабільно
  протягом тижня використання.
- Користувач розуміє, як користуватися продуктом без додаткових
  інструкцій.
